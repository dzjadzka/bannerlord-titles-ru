# Titles RU Localization

Русская локализация для мода **Titles** `v1.0.2` для Mount & Blade II: Bannerlord.

## Требования

- Mount & Blade II: Bannerlord
- Оригинальный мод: `Titles v1.0.2`

## Установка

Рекомендуемый вариант для публикации — отдельный модуль:

1. Скачайте `TitlesRU-v1.0.2-ru1.zip` из GitHub Releases.
2. Распакуйте архив в папку игры так, чтобы получилось:
   `Modules/TitlesRU/SubModule.xml`.
3. Убедитесь, что оригинальный мод `Titles` установлен и включён.
4. Включите `Titles RU Localization` после оригинального `Titles`.

Также в релизе есть `Titles-RU-patch-v1.0.2-ru1.zip` — локальный patch-вариант, который добавляет `RU` прямо в папку оригинального мода.

## Источник

Использован частичный RU pack `Titles RU V1.0.2` как источник кандидатов. Он покрывал `103/115` строк; `12` строк были переведены вручную. Ссылки на отсутствующие preset-файлы из исходного `language_data.xml` не переносились.

## Nexus Mods

Первичная страница Nexus должна быть создана вручную. После создания страницы можно подключить автоматическую загрузку следующих версий через GitHub Actions/Nexus Mods upload action.
