# Titles RU Localization

Русский перевод мода **Titles v1.0.2**.

Переводит настройки, подсказки, уведомления, энциклопедию и отображение титулов в игре.

## Требования

- Оригинальный мод **Titles v1.0.2**

## Установка через Vortex

1. Нажмите **Mod Manager Download** / **Download with Vortex**.
2. Установите мод через Vortex.
3. В лаунчере включите сначала **Titles**, затем **Titles RU Localization**.

## Ручная установка

1. Скачайте архив **TitlesRU-v1.0.2-ru1.zip**.
2. Распакуйте его в папку игры.
3. Должна появиться папка:

```text
Modules/TitlesRU/
```

4. В лаунчере включите сначала **Titles**, затем **Titles RU Localization**.

Архив универсальный: подходит и для Vortex, и для ручной установки. Он ставится отдельным модулем и не заменяет файлы оригинального мода.
