# Titles RU Localization

Русская локализация для мода **Titles v1.0.2**.

## Требования

- Оригинальный мод **Titles**
- Mount & Blade II: Bannerlord

## Установка

Рекомендуемый файл: `TitlesRU-v1.0.2-ru1.zip`.

Распакуйте архив в папку игры. Должна появиться папка:

```text
Modules/TitlesRU/
```

В лаунчере включите оригинальный `Titles`, затем `Titles RU Localization`.

## Проверка

- Покрытие строк: `115/115`
- Missing IDs: `0`
- Extra IDs: `0`
- Placeholder mismatches: `0`
- XML errors: `0`

## Примечание

Перевод основан на частичном RU pack `Titles RU V1.0.2`, дополнен и вычитан вручную. Некорректные ссылки на отсутствующие preset-файлы из исходного pack не переносились.
